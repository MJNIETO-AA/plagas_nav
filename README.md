# plagas_nav
Sistema de gestión para control de plagas desarrollado en Python, Tkinter y MySQL.
# Sistema de Gestión de Control de Plagas

Proyecto desarrollado como parte del servicio social para la empresa
Exterminadora de Plagas Navarro S.A. de C.V.

## Tecnologías usadas
- Python
- Tkinter
- MySQL

## Funcionalidades
- Gestión de clientes
- Gestión de empleados
- Registro de servicios
- Control de pagos
- Inventario
- Generación de reportes
- Sistema de backup
